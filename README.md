# ML_analysis_trading_strategy
Applying ML models for the user to get a future clsoing price
In this code example for the price an ML model will just create a forecast to predict the future price to determine the probability that the trading strategy will lead to future profit or not. 

Both Univarate ML analysis that only uses the closing price and date will be available as options. 
Also, Multivariate ML analyis that uses more than one feature and a more customized model will be available. 

There will be a future strike price with various predictions and a spread factor to assess the validity of the ML run. Refer to the documentation on the ML training method for more information such as this one: https://winedarksea.github.io/AutoTS/build/html/index.html
